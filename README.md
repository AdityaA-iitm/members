# Add your details to members.js

1. Fork the Repo
2. Open `members.js`
3. Make a new entry and fill the required information in correct format.


A sample is given below 

```javascript
    {
        "name" : "null",
        "email" : "null",
        "links" : {
            "github" : "null",
            "linkedin" : "null",
            "instagram" : "null",
            "personal-website" : "null",
        },
        "photo" : "null",
        "shortIntro" : "null",
        "position" : "null",
    }
```
